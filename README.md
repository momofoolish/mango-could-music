#### MusicEverywhere 介绍
##### 为什么开发这款软件？
目前音乐版权分布在各个平台上，我们想听喜欢的歌不得不去下载很多平台，这导致打开很多个软件，而且不能同时去将喜欢的音乐添加到同一个播放列表，非常糟糕的体验，此软件就是为了解决这问题而开发的，用户可以自己导入自己的音源播放音乐。
